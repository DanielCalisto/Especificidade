Especificidade

Especificidade CSS é um tipo de “poder de escolha”, que ajuda os navegadores a decidir qual valor da propriedade CSS terá precedência e será aplicado aos elementos. Geralmente, a especificidade é o culpado por você bater cabeça quando pensa que o valor das propriedades CSS deve se aplicar a um elemento, mas não.

Seletores

Universal Seletor universal é representado por um asterisco, “*”, e casa com todos os elementos do documento.

Seletor por ID Este seletor é utilizado quando desejamos identificar um elemento específico no nosso documento HTML para sua estilização no CSS usando “#”.

Seletor por Classe Um elemento HTML pode ter um ou mais itens definidos em seu atributo de class . O seletor de classe corresponde a qualquer elemento que tenha essa classe aplicada a ele.

Seletor por Elemento Este seletor identifica todas as tags do documento HTML e estiliza no CSS, por exemplo você pode estilizar todas as divs de uma vez só.

Seletor por Pseudo-Classe Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado. Por exemplo, :hover pode ser usado para alterar a cor de um botão quando o usuário passar o cursor sobre ele.

Seletor por Pseudo-Elemento Um pseudo-elemento CSS é uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado. Por exemplo, o pseudo-elemento ::first-line aplica o estilo apenas na primeira linha de um parágrafo.

Calcúlo

Como calcular a especificidade? O cálculo de especificidade é uma combinação desses três tipos de seletores, que gera algo mais ou menos assim: (0, 0, 0) Separadamente, esses números representam a quantidade de ids, classes e elementos usados na formação de um seletor CSS. O digito a esquerda terá maior prioridade que o da direita, ou seja, a primeira posição é a de maior peso, enquanto a terceira posição é a de menos peso. Então será feita uma comparação, de número a número, para decidir qual seletor possui maior especificidade (ou peso/prioridade).

!Important

Uma exceção !important Quando a regra !importanté utilizada na declaração do estilo, substitua qualquer outra declaração feita no CSS, onde quer que esteja na lista de declaração. Contudo, !importantnão tem nada a ver com especificação.
